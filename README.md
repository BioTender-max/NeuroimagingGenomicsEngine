# NeuroimagingGenomicsEngine

Integrate two-photon calcium imaging and Neuropixels electrophysiology for neural population dynamics analysis.

## Features

- Two-photon calcium transient detection and quantification
- Neuropixels spike sorting and unit quality assessment
- Principal component analysis of neural population activity
- Theta band power spectral analysis
- Cross-modal neural activity correlation

## Results

50 exp × 200 neurons; Mean transients=50.1; PC1=33.0%; Theta power=321.3 µV²/Hz

## Usage

```bash
pip install numpy scipy matplotlib
python neuroimaging_genomics_engine.py
```
